# Workout Tracker

This is a long overdue app that I am creating. The sole purpose for this app is to make life in the gym easier. everyone wants to a schedule, everyone wants to track their weights, what targets they want to achieve and what exercises to do on certain days! The idea is to create something fun, and interactive rather than a boring excel spreadsheet, i want to something that the user can personalise, track and edit easily.

# Features 

a secure google auth login
edit days, exercises, weights, sets, and reps
